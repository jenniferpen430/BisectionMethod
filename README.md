# BisectionMethod

Approximate's f(x) = 0 based on input x and tolerance level
## Usage

Example: f(x) = x^3 + x -5

You are to enter the tolerance of 0.0001,
and the interval endpoints Left = 0  and Right =1
and
Left =0 and Right = 2.
